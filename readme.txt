Extend WooCommerce product search functionality with multiple search options and configurable settings.

== Description ==

Enhance your WooCommerce product search with these features:

- Search in product attributes (color, size, etc.)
- Include SKU in search results
- Search custom fields
- Include product categories and tags
- Search product excerpts
- Find products through their variations
- Per-field exact match options
- Configurable settings page
- Multi-language ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/enhanced-search-for-woocommerce` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure settings under WooCommerce â†’ Enhanced Search

== Frequently Asked Questions ==

= What permissions do I need to configure the plugin? =

You need 'manage_woocommerce' capability (typically Shop Manager or Administrator roles).

= Will this affect my site's performance? =

The plugin is optimized for standard WooCommerce stores. For large stores (10,000+ products), consider using a search index plugin.

= Can I contribute translations? =

Yes! Translations are welcome.
